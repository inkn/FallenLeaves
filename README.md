# FallenLeaves
A simulation system of deciduous scene based on Three.js.

### [点击演示](https://inknight.cn/FallenLeaves/index.html)

### 安装

``` shell
git clone git@github.com:inkn/FallenLeaves.git
```
用 Microsoft Edge 浏览器打开 index.html 或者在编辑器里启动服务打开。
